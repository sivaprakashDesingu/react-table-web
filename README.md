# react-table-web

## Install

```bash
npm install --save react-table-web
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-table-web'
import 'react-table-web/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
